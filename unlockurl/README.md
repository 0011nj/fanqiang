# 解封禁闻直连网址


<h2>获得一个直连网址的域名</h2>

<ol type=disc>
<li >点<a href="https://github.com/bannedbook/fanqiang/wiki#jwurl">禁闻代理在线-镜像</a>, 然后点“禁闻代理在线-镜像1” ，即可打开一个禁闻代理网址，把这个网址拷贝下来，比如得到网址：https://ya.dnstogo.xyz/ ，然后我们把前面的https:// 和 后面的一个斜杠去掉，就得到了一个域名：ya.dnstogo.xyz</li>

</ol>
